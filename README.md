# DentAssist Pro — Инструкция по установке

## Что это?
Мобильное веб-приложение (PWA) для врача-стоматолога.
Работает на iPhone и Android без App Store и Google Play.

---

## Шаги для публикации (бесплатно, ~15 минут)

### 1. Создайте аккаунт GitHub
- Зайдите на https://github.com
- Нажмите «Sign up» и зарегистрируйтесь

### 2. Создайте новый репозиторий
- Нажмите «+» → «New repository»
- Название: `dental-assist-pro`
- Выберите «Public»
- Нажмите «Create repository»

### 3. Загрузите файлы
- Нажмите «uploading an existing file»
- Перетащите ВСЕ файлы из этой папки
- Нажмите «Commit changes»

### 4. Создайте аккаунт Vercel
- Зайдите на https://vercel.com
- Нажмите «Sign up» → войдите через GitHub

### 5. Разверните приложение
- Нажмите «Add New Project»
- Выберите репозиторий `dental-assist-pro`
- Framework Preset: **Create React App**
- Нажмите «Deploy»
- Через 2 минуты вы получите ссылку вида: `dental-assist-pro.vercel.app`

---

## Как установить на телефон

### iPhone (Safari):
1. Откройте ссылку в Safari
2. Нажмите кнопку «Поделиться» (квадрат со стрелкой вверх)
3. Выберите «На экран «Домой»»
4. Нажмите «Добавить»
✅ Иконка появится на рабочем столе!

### Android (Chrome):
1. Откройте ссылку в Chrome
2. Нажмите три точки (меню) в правом верхнем углу
3. Выберите «Добавить на главный экран»
4. Нажмите «Добавить»
✅ Иконка появится на рабочем столе!
